# AFG [Anime Funny Game]

AFG is a home project that is being developed TypeScript + Reacr Native.

### How to start

AFG requires [Yarn](https://yarnpkg.com/) + [Expo](https://expo.io) to run.

YARN:
```sh
$ yarn install
$ yarn start
```
